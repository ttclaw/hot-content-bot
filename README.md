# AI 热点内容自动生成服务

## 项目概述
自动抓取全网热点，生成深度分析内容，发布到多平台获取流量收益。

## 技术栈
- Python 3.10+
- Playwright / Requests（抓取）
- Kimi API（内容生成）
- GitHub Actions（定时任务）
- Vercel（部署）

## 目录结构
```
content-service/
├── src/
│   ├── scraper/          # 爬虫模块
│   ├── generator/        # 内容生成
│   ├── publisher/        # 发布模块
│   └── scheduler.py      # 调度器
├── config/
│   ├── sources.json      # 数据源配置
│   └── prompts/          # 生成提示词
├── data/
│   ├── raw/              # 原始数据
│   └── processed/        # 处理后内容
├── logs/                 # 运行日志
└── README.md
```

## 数据源（第一阶段）
- [ ] 微博热搜

## 内容类型
1. 热点快讯（短内容，适合小红书/即刻）
2. 深度分析（长文，适合公众号）
3. 数据可视化（图表，适合多平台）
